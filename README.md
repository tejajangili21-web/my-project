**FlixBus Clone – Bus Ticket Booking Website UI**


This project is a **frontend clone of the FlixBus website** developed using **HTML and CSS**. It provides a modern and responsive user interface for booking bus tickets online. The website includes a navigation menu, search section, service features, travel information, city listings, and footer links.

---

## Technologies Used

* **HTML5** – Structure of the webpage
* **CSS3** – Styling and layout
* **Flexbox** – Responsive alignment
* **CSS Grid** – City and footer layouts
* **Media Queries** – Mobile responsiveness

## Conclusion

The **FlixBus Clone Project** is a frontend web development project that demonstrates HTML and CSS skills by recreating a professional bus ticket booking website. It helps students understand website layouts, Flexbox, Grid, responsive design, and modern UI development concepts. This project can be extended into a complete bus reservation system using JavaScript, PHP, Node.js, and MySQL.
